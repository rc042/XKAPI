XKAPI

A simple GET endpoint for creating XKPasswds

Avaliable at /api

Four different GET parameters:

  • complexity - Changes the complexity of the password. See here for more information
    • default = 5
  • separators - Changes the characters between the words (lorem#ipsum)
    • default = #.-=+_!$*:~?%^&;
  • pattern - Changes how the password is formed.
    • w - lowercase word
    • W - uppercase word
    • s - separator character
    • d - random digit
  • transform - Changes how each word is capitalized
    • alternate - EVERY other WORD is CAPITALIZED
    • uppercase - EVERY WORD IS CAPITALIZED
    • random - every word IS pseudo RANDOMLY CAPITALIZED

Examples

/api = didelphous.benzidine.cynopodous.isocyanate.99

/api?complexity=6 = 02=reality=nonquantitative=remembers=underpaying=pioneys=21

/api?separators=-^ = demayne^catabolized^recommendably^dynastical^41

/api?pattern=wd = sculduddery1

/api?transform=uppercase = ANTIGENE.PALATINATE.TELEOLOGIST.ELECTROCARDIOGRAPH.88

/api?separators=-&pattern=ddswswsdd&transform=random = 18-chevroned-KHUTBAHS-19
